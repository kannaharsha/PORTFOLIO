---

## ✨ Key Features

- **💎 Advanced Tech-Graphics**: Integrated multi-layered tech-rings and scanner sweep animations for a state-of-the-art AI aesthetic.
- **🌌 Ambient Code Layer**: Subtle, drifting fragments of Python, ML, and React code integrated into the background for a "live system" feel.
- **📱 Responsive 3D Interaction**: Interactive 3D project cards with neon-blue grid patterns and glassmorphism.
- **✨ Scroll-Reveal Animations**: Every section features smooth, high-fidelity entry animations powered by Framer Motion.
- **📊 Interactive Skill Matrix**: A categorized expertise section with hover-responsive metrics and tooltips.
- **📨 Integrated Communication**: Fully functional contact system with real-time validation via Web3Forms.

## ⚙️ How to Run Locally

Follow these steps to explore the code or run the portfolio on your environment:

1. **Clone the repository**
   ```bash
   git clone https://github.com/kannaharsha/PORTFOLIO.git
   cd PORTFOLIO
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Spin up the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
